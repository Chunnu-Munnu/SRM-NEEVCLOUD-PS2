DemoGen Pitch
=============

Title
-----
DemoGen: Intelligent Screen Recording, Semantic Narration, and Interactive Walkthrough Generation for NeevCloud

One-Line Positioning
--------------------
DemoGen is an AI-powered, persona-aware demo generation engine that turns a real NeevCloud workflow into a guided, multilingual, shareable interactive product walkthrough.

Problem We Are Solving
----------------------
Today, product demos for platforms like NeevCloud are mostly manual.
A team member has to:
- navigate the portal manually
- record the screen
- write a script
- record a voiceover
- edit the final output
- repeat the whole process whenever the UI changes or a new persona needs a different explanation

This makes product demos:
- time-consuming
- inconsistent
- difficult to scale
- hard to personalize for different user roles

Our Core Insight
----------------
A demo is not just a recording problem.
A demo is a semantic understanding and storytelling problem.

That means a useful demo system should not only capture what happened on the screen. It should understand:
- what action the user is performing
- why that action matters
- how to explain it differently for different personas
- how to package it into a reusable product experience

What DemoGen Does
-----------------
DemoGen helps turn a NeevCloud use case into a complete demo pipeline.

It allows a user to:
- enter an intent such as "show me how to launch a GPU instance for ML training on NeevCloud"
- generate an AI-planned portal flow
- capture the actual product journey through guided live capture
- validate whether each shared screen matches the intended step
- generate contextual narration for every step
- explain the same flow differently for different personas
- localize the walkthrough into multiple languages
- export the result as a PDF or shareable interactive walkthrough

Instead of re-recording full videos every time a flow changes, DemoGen can regenerate the story and walkthrough structure from a new capture.

How DemoGen Works
-----------------
DemoGen works in five layers.

1. Intent Layer
The user enters a goal in natural language.
Example:
"Show me how to launch a GPU instance for ML training on NeevCloud."

DemoGen uses AI to convert that goal into an ordered navigation plan.

2. Flow Planning Layer
The system predicts which NeevCloud pages and actions are needed.
For each step, it defines:
- the action
- the purpose of the step
- the expected UI elements

3. Guided Capture Layer
The user shares the NeevCloud tab.
Instead of blindly recording, DemoGen guides the capture process step by step.
For each planned step, the system can:
- inspect the current shared screen
- compare it against expected elements
- tell the user whether it is the right screen
- recommend what to do next
- capture the frame only when it is suitable

4. Semantic Narration Layer
For each captured frame, DemoGen generates:
- contextual narration
- highlighted UI focus area
- API call mapping if relevant
- code snippet if relevant
- persona-specific explanation

5. Output Layer
The final result is packaged as a polished interactive walkthrough that can be:
- played step by step
- auto-played like a demo sequence
- explored interactively
- exported as PDF
- shared through a link
